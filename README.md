אתר מתכונים – פרויקט צד לקוח ב-React
תיאור הפרויקט:
זהו פרויקט פרונטנד מבוסס React המציג רשימת מתכונים ומאפשר למשתמשים לראות פרטים נוספים על כל מתכון, להוסיף מתכונים חדשים (אם הם מנהלים), ולבצע פעולות שונות בהתאם להרשאות.

פונקציונליות עיקרית
דף מתכונים – מציג את כל המתכונים הקיימים עם תמונה, שם ופרטים כלליים.
דף פרטי מתכון – לחיצה על מתכון תעביר לדף שבו מוצגים כל פרטי המתכון.
הוספת מתכון חדש – משתמשים עם הרשאה יכולים להוסיף מתכון חדש דרך טופס ייעודי.
רשימת משתמשים – מוצגת רק למנהלים, וכוללת את כל המשתמשים במערכת.

מבנה טכנולוגי
React – לבניית ממשק המשתמש.
Redux – לניהול מצב הגלובלי של האפליקציה.
Context API – משמש לניהול הרשאות המשתמשים.
React Router – לניווט בין הדפים השונים באפליקציה.
Axios – לשליחת בקשות לשרת ה-API.
Bootstrap – לעיצוב מהיר ותגובה.

תיקיות עיקריות
components/ – קומפוננטות של ממשק המשתמש (כגון תצוגת מתכון, דף פרטי, טופס הוספה).
redux/
reducers/ – קבצי reducers לכל חלק במצב.
actions/ – קבצי פעולות שמבצעות שליפות/שינויים.
store.js – קובץ יצירת ה-store.
api/axios.js – קובץ שמרכז את קריאות ה-API באמצעות axios.
